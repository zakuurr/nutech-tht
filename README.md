## Installation

- `git clone https://github.com/zakuurr/nutech-tht.git`
- `cd nutech-tht/`
- `composer install`
- `php artisan jwt:secret`
- `npm install`
- `cp .env.example .env`
- `create database`
- Update `.env` change DB_DATABASE to database name
- `php artisan key:generate`
- `php artisan migrate:fresh --seed`
- `php artisan serve`
